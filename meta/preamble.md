# The Primary Distinction

_The act from which the entire system unfolds_

## The Starting Point

Before building a system, one must pose the question it answers. This question cannot be avoided: the very act of avoiding it already answers it.

## Two logics

| | Logic of Capital | Logic of Constitution |
|---|---|---|
| **Subject** | Resource + | Sovereign |
| **Attention** | Valuable because finite | Protected because finite |
| **System** | Subject serves system | System serves subject |

> A neutral position does not exist. Every interaction protocol already applies one of these logics.

## The originating contradiction Ω₀

A human cannot remain awake for more than ~16 hours per day. This is an absolute physiological limit. Digital capital treats this attention as a potentially infinite resource: there is no internal saturation mechanism, no point at which the system says "enough".

This contradiction is self-reproducing: the attempt to deny it already applies one of the two logics to the act of denial.

::formula
∀s ∈ S: F(s) ∈ {Resource(s), Sovereign(s)}
Resource(s) ∩ Sovereign(s) = ∅
::

## NA0 — The single postulate

**Subjecthood is a politically protected good.**

NA0 is not an empirical observation. It is a normative axiom: a _decision_ that subjecthood must be protected. The entire architecture P0–P18 is derived from this decision.

To reject NA0 is to accept that the subject is a resource. To accept NA0 is to accept the necessity of everything that follows from it.

::callout{Consequence}
NA0 is the system's single postulate. All other statements are theorems that follow from NA0 + description of reality.
::
